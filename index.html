소개

이 웹사이트가 무엇을 수집하고, 어떤 방식으로 전시/교육으로 연결하는지 한 번에 보여주는 섹션.

미션

클래식 음악 관련 자료를 체계적으로 수집·정리하고, 이해 가능한 전시/교육 단위로 재구성한다.

  • COLLECT 악보, 공연 프로그램, 레코딩, 인터뷰, 사진, 교육 자료
  • CURATE 주제/시대/작곡가/장르별 구조화
  • LEARN 관람자가 스스로 탐색하고 학습하도록 설계

구성

처음엔 가볍게 시작하고, 필요할 때 기능을 늘릴 수 있게 설계한다.

  • ARCHIVE 소장품 목록 + 메타데이터(출처/연도/권리)
  • EXHIBIT 큐레이션 전시(텍스트·이미지·오디오)
  • EDU 교육 콘텐츠(워크시트/영상/가이드)

기술

GitHub Pages에 올렸을 때 “그냥 돌아가게” 하는 최소 기술 스택. 필요하면 점진적으로 확장.

정적 사이트

HTML/CSS/JS만으로 동작. 배포는 GitHub Pages.

콘텐츠 구조

초기엔 섹션 중심. 나중엔 페이지 분리(컬렉션/전시/교육).

확장 옵션

필요하면 JSON 데이터 + 검색, 또는 Jekyll/Next.js로 전환.

로드맵 예시

  • v0 단일 페이지(현재) — 소개/기술/프로젝트/연락처
  • v1 소장품 페이지 추가 — 카드 목록 + 태그 필터
  • v2 전시 페이지 — 큐레이션 스토리 + 오디오 임베드
  • v3 교육 섹션 — 다운로드 자료 + 짧은 가이드
